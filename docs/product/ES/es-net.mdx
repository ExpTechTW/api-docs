# ES-Net

## 物理規格

| 參數     | 數值         | 單位 |
| :------- | :----------- | :--- |
| 長度     | `70`         | `mm` |
| 寬度     | `35`         | `mm` |
| 高度     | `24`         | `mm` |
| 重量     | `36.8`       | `g`  |
| 防護等級 | `IP10`       | -    |
| 工作溫度 | `-20` ~ `65` | `°C` |

:::danger 溫度使用警告

**外殼材質限制**

- 裝置電子元件經測試可於 `65°C` 環境下正常運作
- 因外殼為 **PLA 材質**，**建議工作溫度上限為 `60°C`**
- 長時間高於 `60°C` 可能導致 PLA 外殼**變形**

:::

| :------- | :---- | :--- |
| 最小電壓 | `4.5` | `V`  |

### NTP 時間同步

| 參數     | 數值    |
| :------- | :------ |
| 校時週期 | `120s`  |
| 時間精度 | `50ppm` |
| 時間誤差 | `±10ms` |

### 模式說明

#### 在線模式

:::tip 推薦模式

- 透過 NTP 校時，通常可提供誤差 `±10ms`
- 時間精度：`50ppm`
- 校時週期：每 `120s` 自動校時一次

:::

#### 離線模式

:::warning

- NTP 同步時間功能關閉
- 透過 `UART` 將資料傳出
- **波形圖時序較不準**

:::

## 感測器規格

### 基本參數

| 參數       | 數值               |
| :--------- | :----------------- |
| 感測器類型 | `MEMS 強震儀`      |
| 感測器軸數 | `三軸 (XYZ)`       |
| 頻率範圍   | `DC-25Hz`          |
| 採樣率     | `50 SPS`           |
| 資料解析度 | `16 bit`           |
| 雜訊密度   | `60 µg/√Hz`        |
| 滿量程     | `2G (約 1961 gal)` |

### 濾波器

內建濾波器包含時域濾波及頻域濾波：

:::info 時域濾波 (BPF)

- **HPF**：`11 級 IIR`，約 `1Hz` 截止頻率
- **LPF**：`11 級 IIR`，約 `10Hz` 截止頻率

:::

:::info 頻域濾波

- `0.5Hz` 以下衰減
- `10Hz` 以上衰減
- 週期效應補償：`√(1/f)`

:::

## 通訊規格

### 有線通訊

| 參數     | 數值                       |
| :------- | :------------------------- |
| 通訊介面 | `USB Type-C 串列埠 (UART)` |
| 傳輸速率 | `115200 bps`               |
| 數據格式 | `二進位`                   |

:::info 輸出數據

- `三軸加速度 (gal)`
- `三軸加速度-濾波 (gal)`
- `JMA 計測震度`
- `PGA(a)`

:::

### 無線通訊 (Wi-Fi)

| 參數         | 數值            |
| :----------- | :-------------- |
| 頻段         | `2.4 GHz`       |
| 標準         | `802.11 b/g/n`  |
| 天線輸出功率 | `20 dBm` (最高) |
