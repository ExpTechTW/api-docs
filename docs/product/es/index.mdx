# 地震儀 (ES 系列)

### 概述

ES 系列地震儀是一款以「低成本、高靈敏」為核心理念所設計的高精度地震監測儀器。整體系統採用高性能三軸 MEMS 加速度計作為感測核心，能夠同時量測三個方向的地面加速度變化，並提供穩定且高解析度的地震波形資料。裝置外型輕巧緊湊，重量僅 36.8 公克，便於攜帶與安裝，可靈活部署於多種場域。
ES 系列特別適合用於地震教育與教學演示、學術研究、地震監測網建置、以及地震學資料分析等應用場景。其設計理念強調「平價普及」，讓更多教育單位與研究機構能以合理成本取得高品質的地震觀測資料，進而推動地震科學的推廣與研究發展。

### 功能

- 高精度三軸 MEMS 加速度計，可量測三個方向的加速度變化。
- 可透過 USB Type-C 串列埠 (UART) 連接電腦，輸出高達 50 SPS 的採樣率及 16 位元解析度的資料。
- 可透過 NTP 同步時間，使資料時間精度達到毫秒級。
- 超低功耗設計，待機功耗僅 0.8 瓦特，適合長期監測。
- 搭配配套軟體 QuakeWatch-Pro ，開箱及用。
