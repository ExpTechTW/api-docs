| 模式     | 電壓   | 功耗  | 單位 |
| :------- | :----- | :---- | :--- |

- **離線模式**：NTP 同步時間功能關閉
  :::

## 感測器規格

### 基本參數

| 參數       | 數值               |
| :--------- | :----------------- |
| 感測器類型 | `MEMS 強震儀`      |
| 感測器軸數 | `三軸 (XYZ)`       |
| 頻率範圍   | `DC-25Hz`          |
| 採樣率     | `50 SPS`           |
| 資料解析度 | `16 bit`           |
| 雜訊密度   | `60 µg/√Hz`        |
| 滿量程     | `2G (約 1961 gal)` |

### 濾波器

內建濾波器包含時域濾波及頻域濾波：

**時域濾波 (BPF)**

- HPF：`11 級 IIR`，約 `1Hz` 截止頻率
- LPF：`11 級 IIR`，約 `10Hz` 截止頻率

**頻域濾波**

- `0.5Hz` 以下衰減
- `10Hz` 以上衰減
- 週期效應補償：`√(1/f)`

## 通訊規格

### 有線通訊

| 參數     | 數值                       |
| :------- | :------------------------- |
| 通訊介面 | `USB Type-C 串列埠 (UART)` |
| 傳輸速率 | `115200 bps`               |
| 數據格式 | `二進位`                   |

**輸出數據**

- `三軸加速度 (gal)`
- `三軸加速度-濾波 (gal)`
- `JMA 計測震度`
- `PGA(a)`

### 無線通訊 (Wi-Fi)

| 參數         | 數值            |
| :----------- | :-------------- |
| 頻段         | `2.4 GHz`       |
| 標準         | `802.11 b/g/n`  |
