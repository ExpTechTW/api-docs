# 自定義事件註冊

除了 TREM 本身，擴充還可以分發自己的事件。

:::tip

**自定義事件**是擴充之間，傳遞訊息的一種方式。

:::

## 用法

事件發射

```js
const emit = (event, data) => TREM.variable.events.emit(event, data);
emit("test", { data: "test" });
```

事件接收

```js
const event = (event, callback) => TREM.variable.events.on(event, callback);
event("test", (data) => {
  console.log(data); // { data: "test" }
});
```
