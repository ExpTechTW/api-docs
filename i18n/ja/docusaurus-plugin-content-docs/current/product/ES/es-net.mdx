# ES-Net

## 物理仕様

| パラメータ     | 数値         | 単位 |
| :------------- | :----------- | :--- |
| 長さ           | `70`         | `mm` |
| 幅             | `35`         | `mm` |
| 高さ           | `24`         | `mm` |
| 重量           | `36.8`       | `g`  |
| 保護等級       | `IP10`       | -    |
| 動作温度       | `-20` ~ `65` | `°C` |

:::danger 温度使用に関する警告

**外殻材質の制限**

- デバイスの電子部品は `65°C` 環境下での長時間正常動作がテスト済み
- **PLA 材質**の外殻のため、**推奨動作温度上限は `60°C`**
- `60°C` を超える長時間使用は PLA 外殻の**変形**を引き起こす可能性があります

:::

### NTP 時刻同期

| パラメータ   | 数値    |
| :----------- | :------ |
| 同期周期     | `120s`  |
| 時刻精度     | `50ppm` |
| 時刻誤差     | `±10ms` |

### モード説明

#### オンラインモード

:::tip 推奨モード

- NTP 時刻同期により、通常 `±10ms` の誤差を提供可能
- 時刻精度：`50ppm`
- 同期周期：`120s` ごとに自動同期

:::

#### オフラインモード

:::warning

- NTP 時刻同期機能は無効
- `UART` 経由でデータを送信
- **波形タイミングの精度が低下**

:::

## センサー仕様

### 基本パラメータ

| パラメータ     | 数値               |
| :------------- | :----------------- |
| センサータイプ | `MEMS 強震計`      |
| センサー軸数   | `三軸 (XYZ)`       |
| 周波数範囲     | `DC-25Hz`          |
| サンプリングレート | `50 SPS`       |
| データ分解能   | `16 bit`           |
| ノイズ密度     | `60 µg/√Hz`        |
| フルスケール   | `2G` (約 1961 gal) |

### フィルター

内蔵フィルターには時間領域フィルターと周波数領域フィルターが含まれます：

:::info 時間領域フィルター (BPF)

- **HPF**：`11段 IIR`、約 `1Hz` カットオフ周波数
- **LPF**：`11段 IIR`、約 `10Hz` カットオフ周波数

:::

:::info 周波数領域フィルター

- `0.5Hz` 以下で減衰
- `10Hz` 以上で減衰
- 周期的効果補償：`√(1/f)`

:::
