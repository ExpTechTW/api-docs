## Physical Specifications

| Parameter      | Value      | Unit |
| :------------- | :--------- | :--- |
| Length         | `70`       | `mm` |
| Width          | `35`       | `mm` |
| Height         | `24`       | `mm` |

| Protection     | `IP10`     | -    |
| Operating Temp | `-20` ~ `65` | `°C` |

:::danger Temperature Usage Warning

**Enclosure Material Limitation**

- Electronic components have been tested to operate normally at `65°C`
- Due to **PLA material** enclosure, **recommended operating temperature limit is `60°C`**
- Prolonged exposure above `60°C` may cause PLA enclosure **deformation**

:::

### NTP Time Synchronization

| Parameter        | Value    |
| :--------------- | :------- |
| Sync Period      | `120s`   |
| Time Accuracy    | `50ppm`  |
| Time Error       | `±10ms`  |

### Mode Description

#### Online Mode

:::tip Recommended Mode

- NTP time synchronization enabled, typically provides error of `±10ms`
- Time accuracy: `50ppm`
- Sync period: Auto-sync every `120s`

:::

#### Offline Mode

:::warning

- NTP time synchronization disabled
- Data transmission via `UART`
- **Waveform timing is less accurate**

:::

| Full Scale        | `2G` (approx. 1961 gal) |

### Filter

Built-in filter includes time-domain and frequency-domain filtering:

:::info Time-Domain Filter (BPF)

- **HPF**: `11-stage IIR`, approximately `1Hz` cutoff frequency
- **LPF**: `11-stage IIR`, approximately `10Hz` cutoff frequency

:::

:::info Frequency-Domain Filter

- Attenuation below `0.5Hz`
- Attenuation above `10Hz`
- Periodic effect compensation: `√(1/f)`

:::
